Scale free network:

  • Scale free network based on Power- Law distribution
  • If distribution in straight line on log-log plot so it suggests a power law distribution and consider as scale free network
  • Plot a degree distribution on a logarithmic scale

Small world network:

  • Having property of high clustering and short average path length
  • Nodes tends to be highly interconnected
  • It takes only few steps to reach from one node to another in the network
  • Calculate average clustering coefficient and average shortest path length
    • Average clustering coefficient: it measures how tightly connected a node with its neighbor. High clustering tells presence of clusters
    • Average shortest path length: it measures how easily info spread through network. Shortest average path length tells small world network

Homophily network:

  • This is a special type of network in which nodes make bonds with similar other nodes which results in similar properties among neighbors.

Interpretation:

  • Degree distribution follows power law on log-log plot, network shows few nodes with higher degree and many nodes with low degree, it will be SCALE FREE NETWORK
  • High clustering and having shortest average path length, it will be SMALL WORLD NETWORK

ASSORTATIVITY:

  • Assortativity is used to measure the preference of nodes inside the network to connect to nodes with similar or dissimilar attributes
  • Helps to understand the HOMOPHILY or HETEROPHILY in a network.

K-Core:

  • K-core is a concept that shows the connectivity in a network
  • A k-core is maximum subnetworks in which all nodes have degree of at least k
  • Higher the value of k-core output represents more central and well-connected regions of the networks.
  • You can replace the edges with your own network's structure and choose different k values to observe the different k-cores of the graph.

DENSITY:

  • Density measures the ratio of the actual number of edges in a network to the total possible number of edges in that network.
  • It gives an indication of how well-connected the network is
  • The density value will be between 0 and 1
  • A density of 1 would mean that all possible edges are present,while a density of 0 would indicate a completely disconnected graph

Concepts to Analyze Social Networks:

    Closeness:

    • The level of closeness to all nodes in the network and quantifies how fast information will spread to other nodes
    • The higher the closeness centrality value of a node, the closed it is to other nodes in terms of shortest paths

    Betweenness:

    • The amount of short path going through the node and measures the extent to which a node lies on the shortest path between nodes in network
    • Nodes with higher betweenness centrality values are important for maintaining communication and information flow in the network

    Degree:

    • The amount of neighbors of the node
